# Этот документ описывает как должен выглядеть "файл отчета о факторах внешней среды" 

1) файл должен содержать следующие разделы:
- Введение
- Методология
- Результаты
- Выводы

2) файл должен быть в формате PDF и содержать текст, графики и таблицы, которые помогут понять факторы внешней среды, влияющие на стратегическое планирование.
